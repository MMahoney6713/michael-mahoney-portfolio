"On Purpose" - a full-stack capstone project with Springboard

November 30, 2021

Background

Github repository is available here, and the app is hosted on Heroku, here.

This is a project I’m thrilled about. Not only do I have the chance to build something from the ground up, but I am also building an application that is directly useful to me and I think could be useful to others as well. Inspired by concepts from “The Seven Habits of Highly Effective People”, by Stephen Covey, as well as my own keenness to plan things out over the long term, I designed an application that equips users to do long-term planning in a more purposeful way. The basic functionality of the application empowers a user to create their life mission - the qualities that they want their life to be characterized by - and to break it down into goals across weeks and months at a time.

Until now this has been something that I’ve tediously built for myself every year using a collection of google docs. As a web application, though, the program builds the calendars for me, and will allow the saving/editing of my mission statement, mission-minded goals, and milestones. Because it’s a web application, I can then iteratively add further functionality as I want to in the future. Say goodbye, google docs!

For this project, I focused on learning a new back-end framework - Django. With extensive Flask/Python experience, I wanted to build a web app from scratch using Django to better understand some of the differences that exist between the two frameworks.

I have received a lot of encouraging feedback from my mentor regarding my performance in the course and my comfort with the Flask materials, so I saw this capstone as the perfect opportunity to challenge myself by learning Django. Django is not taught in the course, but I know that Django is very popular in the Python/Web Development world and I wanted to better understand the differences between frameworks.
